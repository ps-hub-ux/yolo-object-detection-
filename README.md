# Object Detection using YOLOv8

## 📌 Project Overview
This project performs real-time object detection using YOLOv8 and OpenCV.

## 🚀 Features
- Real-time webcam detection
- Detects multiple objects (person, phone, bottle, etc.)
- Displays bounding boxes with labels

## 🛠️ Technologies Used
- Python
- YOLOv8 (Ultralytics)
- OpenCV

## ▶️ How to Run
```bash
python3 yolo_detection.py
